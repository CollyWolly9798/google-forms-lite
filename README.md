🛠️ Getting Started
Clone the repository:

bash
git clone https://github.com/CollyWolly9798/google-forms-lite.git
Navigate to the project directory:

bash
cd google-forms-lite
Install dependencies:

bash
pnpm install
Start the development server:

bash
pnpm run dev
